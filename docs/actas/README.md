# Actas

Aquí se guardan minutas, acuerdos y registros de reuniones del proyecto.

Cada acta debe incluir como mínimo:
- fecha
- participantes
- temas tratados
- acuerdos tomados
- próximos pasos

Los acuerdos funcionales vigentes sobre el flujo operativo son fuente de verdad para decisiones de negocio y prevalecen sobre cualquier otro documento en caso de conflicto operativo.
